# E-Commerce Analysis Summary (Data Analytics + Business Analytics Focused)

This project analyzes the Brazilian Olist E-Commerce dataset to uncover patterns in customer purchasing behavior, sales trends, and product performance. Using SQL and Python, the analysis explores all 11 relational tables to extract actionable business insights.

Key Features

- Customer Geography: Visualized customer locations to identify high-demand regions.
- Sales Trends: Tracked monthly sales and order counts by product category.
- Customer Behavior: Calculated order frequency, total spend, and average days between purchases.
- Customer Satisfaction: Analyzed review scores and generated word clouds from comments to detect issues.
- RFM Segmentation: Classified customers into actionable segments (e.g., Champions, Hibernating, Lost) based on Recency, Frequency, and Monetary scores.
- Visualization: Used Matplotlib, Seaborn, and Folium to highlight trends and patterns.

Insights
- Delivery delays correlate with lower review scores.
- Most customers purchase only once, but loyal segments drive significant revenue.
- RFM segmentation enables targeted retention strategies and marketing optimization.
- This analysis demonstrates the use of SQL joins, window functions, aggregations, and Python data visualization to uncover meaningful insights for e-commerce strategy.
